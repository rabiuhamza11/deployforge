# DeployForge / FluxDeploy

Multi-platform deployment engine for the Harz ecosystem.

## Overview
Intelligent deployment orchestration engine that automates releases across GitHub, Vercel, Render, Netlify, and Railway from a single dashboard.

## Supported Platforms
- GitHub (repo management)
- Vercel (frontend / Next.js)
- Render (backend / Node.js)
- Netlify (static sites)
- Railway (containerized apps)

## Features
- One-click deploy to all 5 platforms
- GitHub repo creation and push automation
- Unified deployment dashboard (FluxDeploy v4)
- Deploy history tracking via DeployTask entity
- Secure env var injection per deployment
